# KALQY

> Consolidated **KALQY Foundational Stage Game Framework (Phase 1)** aligned with **NEP 2020**, kinesthetic learning, Gurukul-inspired education, and KALQY's positioning as a movement-based learning platform.

## NEP-Aligned Kinesthetic Learning Platform

**Foundational Stage (Age 3-6 | Preschool, LKG & UKG)**

## Vision

To create India's first AI-powered kinesthetic learning ecosystem where children learn through movement, play, exploration, storytelling, and real-world problem-solving.

Inspired by:

- NEP 2020
- Foundational Literacy & Numeracy Mission
- Experiential Learning
- Gurukul Principles
- Multiple Intelligences
- Physical + Cognitive Development

# AGE 3-4 (PRESCHOOL)

**Development Objective:** Learn through movement and sensory exploration.

**Academic : Physical Ratio:** 20 : 80

## Game 1: Animal Walk Adventure

**NEP Outcome:** Gross Motor Development

**Gameplay:**

Kalqy visits a jungle. Children imitate:

- Frog
- Rabbit
- Elephant
- Duck

**Movements Tracked:**

- Jumping
- Crawling
- Squatting
- Walking

**Skills Developed:**

- Balance
- Coordination
- Body Awareness

---

## Game 2: Colour Hunt

**NEP Outcome:** Colour Recognition

**Gameplay:**

Kalqy asks: "Find something RED." Children touch floating red objects.

**Movements:**

- Reach
- Point
- Side Step

---

## Game 3: Shape Catcher

**NEP Outcome:** Shape Recognition

**Gameplay:**

Catch only circles. Avoid triangles.

**Movements:**

- Reach
- Swipe
- Jump

---

## Game 4: Sound Safari

**NEP Outcome:** Listening Skills

**Gameplay:**

Hear sound. Select matching animal.

**Movements:**

- Point
- Touch
- Reach

---

## Game 5: Dance with Kalqy

**NEP Outcome:** Physical Development

**Gameplay:**

Follow Kalqy's dance.

**Movements:**

- Jump
- Clap
- Turn
- Wave
